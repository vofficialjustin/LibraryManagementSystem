# Library Management System

## Overview
This project is a Java console application that simulates a simple library system. It is designed to practice data structures and object-oriented programming concepts such as classes, interfaces, iterators, comparators, and collections.

---

## Features
- Add books to a library
- View all books
- Search for books
- Sort books (by title, author, etc.)
- (future) checkout / return system

---

## Technologies Used
- Java
- Gradle
- JUnit 5

---

## Concepts Practiced
- Object-Oriented Programming (OOP)
- Interfaces
- Comparable & Comparator
- Iterators
- Collections (ArrayList, HashMap, etc.)
- Unit Testing (JUnit)
